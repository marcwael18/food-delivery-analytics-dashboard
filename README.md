# Food Delivery Analytics Dashboard

## Overview

This project is an interactive dashboard developed using R and Shiny to analyze food delivery operations and identify factors affecting delivery efficiency. The dashboard combines data preprocessing, machine learning techniques, and business analytics to generate actionable insights from delivery data.

## Features

* Data cleaning and preprocessing
* Missing value handling
* Outlier detection and treatment
* Interactive data visualization
* K-Means clustering analysis
* Apriori association rule mining
* Delivery performance evaluation

## Technologies Used

* R
* Shiny
* dplyr
* ggplot2
* cluster
* arules
* plotly

## Project Objectives

* Analyze food delivery performance
* Identify factors contributing to delivery delays
* Discover hidden patterns in operational data
* Support data-driven decision making
* Generate business recommendations for improving efficiency

## Business Insights

* Traffic conditions significantly influence delivery times.
* Weather conditions can affect delivery performance.
* Different vehicle types perform differently under varying operational conditions.
* Data-driven analysis can help optimize delivery operations and reduce delays.

## Key Analysis Areas

* Traffic Conditions
* Weather Impact
* Vehicle Type Performance
* Delivery Distance
* Delivery Time Analysis
* Operational Efficiency
